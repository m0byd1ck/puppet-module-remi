# Configure remi repository

## About

The following repo will be setup and enabled by default:

  * remi
  * remi-php56

Other repositories that will setup but disabled

  * remi-php55
  * remi-php70
  * remi-debuginfo
  * remi-test
  * remi-test-debuginfo

## Usage
### Simple Usage

    include remi
